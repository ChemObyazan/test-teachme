# Проект автоматизации тестирования для сайта Траектория

## Описание
Проект автоматизации тестирования, включающий UI и API тесты для сайта [Траектория](https://www.traektoria.ru/). Проект реализован с использованием Page Object Model (POM).

### Структура проекта:
- **pages/**: Классы для различных страниц сайта.
- **locators/**: Локаторы для элементов страниц.
- **tests/**: Тесты для различных сценариев, включая UI и API тесты.
- **helpers/**: Вспомогательные функции и классы для проверки элементов.
- **conftest.py**: Настройка фикстуры для браузера.
- **requirements.txt**: Список зависимостей для установки.
- **makefile**: Сценарии для запуска тестов и отчетов.

#### Установка и запуск:
1. Клонировать репозиторий:
git clone <repository-url>

2. Установка зависимостей:
pip install -r requirements.txt

3. Запустить все тесты.
make test_all

4. Запустить UI тесты:
make test_ui

5. Запустить API тесты:
make test_api

6. Сгенерировать отчёт Allure
make allure_report
make test_allure

6.1 **Организация работы с Allure**:
Для генерации отчетов Allure, нужно убедиться, что Allure установлен в системе.

##### Установка Allure:
choco install allure
